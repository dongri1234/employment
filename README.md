项目下载和运行
----

- 拉取项目代码
```bash
git clone https://github.com/tangyuzhe/secondclass-admin.git
cd ant-design-pro-vue
```

- 安装依赖
```
npm install
```

- 开发模式运行
```
npm run serve
```

- 编译项目
```
npm run build
```

- Lints and fixes files
```
npm run lint
```
